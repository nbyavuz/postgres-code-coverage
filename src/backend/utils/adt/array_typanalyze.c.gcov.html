                                177                 :                :  * The algorithm used is Lossy Counting, as proposed in the paper "Approximate

                                181                 :                :  * http://www.vldb.org/conf/2002/S10P03.pdf

                                183                 :                :  * The Lossy Counting (aka LC) algorithm goes like this:
